# AI-Driven Network Intrusion Detection System

## Project Overview

This project is an AI-powered Network Intrusion Detection System (NIDS) developed using Python. It analyzes network traffic from packet capture (PCAP) files and identifies suspicious activities using machine learning.

## Features

- Network traffic analysis
- Packet counting by IP address
- Port scan detection
- AI-based anomaly detection using Isolation Forest
- Intrusion alert generation
- CSV report generation
- Traffic visualization using graphs

## Technologies Used

- Python
- Scapy
- Pandas
- Matplotlib
- Scikit-Learn

## Project Files

- project.py – Main intrusion detection system
- bruteforce.py – Brute-force attack simulation
- dos.py – DoS attack simulation
- requirements.txt – Required Python libraries

## Output

- network_features.csv
- ids_results.csv
